## ⚙️ Cómo Ejecutar

```bash
# Clonar el repositorio
git clone https://github.com/kherreracentellas-dotcom/portafolio-1.git
cd portafolio-1

# Abrir directamente en navegador
# Solo necesitas un navegador, no hay build process
open index.html
```

> **Nota:** El portafolio usa contenido dinámico cargado desde `content.json`. Para desarrollo local, puedes usar un servidor local:
> ```bash
> npx serve .
> # o
> python -m http.server 8000
> ```
